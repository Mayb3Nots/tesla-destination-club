# Tesla Destination Club ⚡

Open source queue management for Tesla destination chargers. Because waiting for a plug shouldn't be a sport.

## Features

- 🗺️ **Nearby Chargers** — Browse TDC locations with live status
- 📊 **Real-time Slots** — See which chargers are open, charging, or offline
- 📋 **Virtual Queue** — Join the queue without leaving your car
- 🔔 **Live Updates** — Get notified when it's your turn
- 📱 **Mobile-first** — Designed for use on the go

## Tech Stack

- **Web:** SvelteKit 5 + Tailwind CSS + TypeScript
- **Hosting:** Cloudflare Pages
- **Mobile:** Flutter (planned)
- **Backend:** TBD

## Dev

```bash
yarn install
yarn dev
```
