# Tesla Destination Club ⚡

Open source queue management for Tesla destination chargers. Because waiting for a plug shouldn't be a sport.

## What

TDC helps Tesla owners queue for free destination chargers at hotels, malls, restaurants, and more. No more circling the parking lot.

## Features (Planned)

- 🔍 Browse destination charger locations
- 📋 Join virtual queues with real-time position tracking
- 🔔 Get notified when it's your turn
- 🔌 Check in / check out to keep the queue moving
- 📊 Charger status (available, charging, waiting)
- 📱 Mobile-friendly (Flutter app later)

## Tech Stack

- **Web:** SvelteKit + Tailwind CSS + TypeScript
- **Mobile:** Flutter (planned)
- **Backend:** TBD (considering Firebase / Supabase)

## Dev
